# 炉石传说自动对战 Bot 任务清单

## 1. 当前范围

- 运行环境固定为 Windows 前台截图 + 前台输入
- 当前支持两个固定分辨率 profile：`1600x900`、`1440x900`
- 当前仍不支持动态分辨率适配
- 游戏模式为窗口化，窗口目标位置为 `(20, 20)`
- 目标模式为休闲模式
- 通过全局热键 `F8` 停止

## 2. 当前运行方式

- 主程序：
  - `python -m bot.main --deck-index 1 --resolution 1600x900`
  - `python -m bot.main --deck-index 1 --resolution 1440x900`
- 采样工具：
  - `python -m bot.sample_main --tag in_battle --deck-index 1 --resolution 1600x900`
  - `python -m bot.sample_main --tag in_battle --deck-index 1 --resolution 1440x900`

说明：

- `deck_index` 范围为 `1-9`
- `match_count` 已移除

## 7. 当前交接重点

手牌检测：

- 已不再使用旧的超宽连通域等宽切块
- 当前主链路为：绿色掩码 -> 原始候选 -> 宽连体拆峰值 -> 去重 -> 可出评分
- 阈值在 `bot/regions/*.yaml` 的 `hand_detection` 段

场景与恢复：

- `bot/vision/scene.py` 中 `confirm` 现在先归类为 `confirm_dialog`
- `bot/runtime.py` 负责将其结合上下文归一化为：
  - `match_error`
  - `result`
  - `result_continue`
- 实机调试时应优先观察归一化原因日志
